# Moomingle - Product Overview

Moomingle is a "Tinder-style" mobile marketplace for buying and selling cattle and buffalo in India. Users can swipe through livestock listings, match with sellers, and chat to negotiate purchases.

## Core Features

- **Swipe-based browsing**: Card-based UI for discovering cattle/buffalo listings with like/pass gestures
- **AI breed classification**: ML-powered breed identification from photos (supports 10 Indian breeds)
- **Seller hub**: Dashboard for managing listings, viewing performance, and boosting visibility
- **In-app chat**: Real-time messaging between buyers and sellers after matching
- **Verification system**: Verified badge for authenticated livestock listings

## Target Market

Indian cattle and buffalo traders, focusing on popular breeds:
- Buffalo: Murrah, Jaffarbadi, Mehsana, Bhadawari, Surti
- Cattle: Gir, Kankrej, Sahiwal, Ongole, Tharparkar

## Design Language

- Warm, earthy color palette (browns, tans, cream backgrounds)
- Primary color: `#8B5A2B` (brown), Background: `#F5E0C3` / `#E8B888`
- Material Design 3 with rounded corners (32px cards, 20px buttons)
- Clean, modern UI inspired by dating apps
